Детальний опис

Клас Manager - це об'єкт-обгортка над іншими об'єктами програми. Він призначений для створення інших об'єктів програми, які мають виконувати основні задачі калькулятора. Об'єкт класу створюється при ініціалізації програми

Компонентні дані

AlgebraicOperations Manager::a_operations

Об'єкт класу AlgebraicOperations призначений для виконання основних арифметичних операцій над числами, зокрема: додавання, віднімання, множення, ділення, зміна знака, отримання модуля числа, його піднесення до квадрату та добування квадратного кореня.

InputManager Manager::i_manager

Об'єкт класу InputManager призначений для проведення валідації введених чисел та їх зберігання

NumPanel Manager::n_panel

Об'єкт класу NumPanel призначений для керування введенням чисел. Він здійснює додавання обраних чисел до поля для введення, які в подальшому використовуються іншими об'єктами

TrigonometricOperations Manager::t_operations

Об'єкт класу TrigonometricOperations призначений для виконання основних тригонометричний операцій над дійсними числами, а саме: отримання синуса, косинуса, тангенса та котангенса аргумента, а також переведення дійсного числа із градусів у радіани.
